# План разработки (Hebrew Verbs Bot TMA)

## Фаза 1: Инициализация и базовая структура
- [ ] Настроить проект (React/Vite или чистый JS/HTML/CSS).
- [ ] Интегрировать Telegram Mini Apps (TMA) SDK.
- [ ] Настроить базовый роутинг (Главная, Викторина, Статистика).

## Фаза 2: Работа с данными и базой
- [ ] Подготовить JSON со словарем глаголов (с огласовками/никудот) и разбить по группам/биньянам.
- [ ] Настроить БД (SQLite) для сохранения прогресса (telegram_id, word_id, status: learned/learning).
- [ ] Логика выборки слов для изучения (алгоритм интервальных повторений или простая очередь).

## Фаза 3: Викторина (Quiz)
- [ ] UI викторины: Вопрос на русском -> 4 варианта на иврите.
- [ ] Проверка правильности и обновление прогресса в БД.
- [ ] Экран результатов.

## Фаза 4: ИИ-помощник (Контекстное меню)
- [ ] Кнопка "Спросить ИИ" в карточке слова.
- [ ] Интеграция API недорогой модели (gpt-4o-mini).
- [ ] Промпт: пример использования, множественное число, объяснение биньяна.

## Фаза 5: Дизайн и темы
- [ ] Создать минималистичный дизайн.
- [ ] Смена цветовой палитры в зависимости от группы глаголов или биньяна.
