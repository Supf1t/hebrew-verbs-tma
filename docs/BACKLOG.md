# Backlog

1. **Статистика по группам:** При выборе группы в выпадающем списке показывать прогресс только для этой группы (например, "Выучено 2 / 16").
2. **Озвучка (TTS):** Добавить иконку динамика рядом со словом на иврите. По клику воспроизводить слово через `window.speechSynthesis`.
3. **Мой словарь (Отдельный экран):** Список всех выученных и невыученных слов (иврит + перевод), чтобы можно было следить за успехами.
4. **Генерация групп через ИИ:** Текстовое поле/кнопка, куда можно написать промпт (например, "10 глаголов путешествий"), ИИ генерирует JSON слов, и они добавляются в текущую сессию/базу.
